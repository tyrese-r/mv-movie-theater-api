# movie-theater-api
Movie Theater Project for Express Week Day 5

Each user can have many shows

# What do I need?

## Routes


### Users
`GET` all users<br>
`GET` one user<br>
`GET` all shows watched by a user (user id in req.params)<br>
`PUT` update and add a show if a user has watched it

### Shows
`GET` all shows<br>
`GET` one show<br>
`GET` shows of a particular genre (genre in req.params)<br>
`PUT` update rating of a show that has been watched<br>
`PUT` update the status of a show<br>
`DELETE` a show<br>
